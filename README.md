
## Configuration.
    vi ./config.py

## Check server status.
    ./bes.sh status

## Start the server.
    ./bes.sh start
  
  Note: You can provide http or https to start server on http or https respectively.
        For example: ./bes.sh start http
        By default it starts on https.

## Stop the server.
    ./bes.sh stop
